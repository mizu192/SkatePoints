# SkatePoints

## どんなアプリか

フィギュアスケートの構成を考えて、点数を計算できるアプリです。

ジャンプやコンビネーションを追加すると、TES（技術点）とPCS（演技構成点）をもとに合計点が出ます。

---

## 作った理由

現在フィギュアスケート選手で、自分が大会の際に自分がどれくらいの点を目標にする際毎回手計算が必要で、いろいろな構成を試してこの構成だと何点ぐらいを出すことができるのか簡単に調べることができたらすごく便利だと感じ作成しました。
また、アプリ開発の経験がないので、aiにいろいろ相談しながら、また本を読みながら学習を進めました。

---

## できること

* 要素の追加
* GOE（出来栄え点）の入力と自動計算
* PCS（3項目）の入力と自動計算
* 合計点の自動計算

---

## 使い方

1. プログラム名を入力
2. ジャンプを選ぶ（単発 or コンビ）
3. GOEを入力して追加
4. PCSを入力
5. 合計点が表示される

---

## 起動方法

自分のPCで実行する場合
コマンドプロンプトで、
cd skating_points
python3 app.py
を実行すると開けます。

または、このgithubのリンクからも開けます。
